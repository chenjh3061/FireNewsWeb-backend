# 火灾信息管理平台后端

本系统是一个消防信息管理平台的后端，
主要包括用户、文章等模块。

## 技术选型：
- 开发环境JDK1.8,  Java8， SpringBoot2.7.4
- MySql 8.2
- Sa-Token轻量级权限认证框架
- ElasticSearch 搜索引擎
- Mybatis-Plus 数据库操作依赖
- Hutool 工具库
- lombok 注解
- Knife4j 接口文档
- 接入智谱 AI GLM-4模型
